# modLinePlot Module

Shiny module which works together with [`modLinePlot`](modLinePlot.md)
to produce a line plot tab

## Usage

``` r
modLinePlotUI(id, presets = NULL)
```

## Arguments

- id:

  id of the filter

- presets:

  Optional list of presets for quick variable selection
