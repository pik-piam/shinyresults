# Get Variables for Preset

Extract the list of variables from a specific preset.

## Usage

``` r
getPresetVariables(config, presetId)
```

## Arguments

- config:

  Variable configuration list from loadVariableConfig()

- presetId:

  ID of the preset to extract variables from

## Value

List of variable configurations, each with 'variable' and 'plot_type'
fields
