# modDashboard UI Module

Shiny module which works together with [`modDashboard`](modDashboard.md)
to produce a dashboard with pre-configured plots based on variable
presets.

## Usage

``` r
modDashboardUI(id, presets = NULL)
```

## Arguments

- id:

  id of the module

- presets:

  Named list of presets from variable configuration (not used, kept for
  compatibility)
